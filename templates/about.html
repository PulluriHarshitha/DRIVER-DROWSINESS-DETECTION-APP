About Smart Driver Monitoring System
🚗 Project Overview

The Smart Driver Monitoring System is an intelligent safety solution designed to detect driver drowsiness and yawning in real-time using computer vision. The system analyzes facial landmarks from live webcam feed to identify signs of fatigue and issues instant audio alerts to prevent accidents.

🧠 Technologies Used
  • Python, OpenCV, Dlib, Imutils for facial landmark detection
  • Flask for web application and dashboard
  • Matplotlib for data visualization
  • Pygame for sound alerts
  • Bootstrap 5 for responsive, modern web UI
🎯 Key Features
  • Real-time drowsiness and yawn detection
  • Instant sound alerts for driver safety
  • Automatic data logging and live analytics dashboard
  • Fully responsive and visually engaging design
